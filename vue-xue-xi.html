一. Vue实例

1.1 声明式渲染

Vue.js的核心是一个允许采用简洁的模板语法来声明式地将数据渲染进 DOM 的系统： 

 <div id="app">
     {{content}}
 </div>
var app = new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue!'
  }
})

1.2 Vue生命周期

 var vm=new Vue({
        el:'#app',
        template:"<div>hello world</div>",
        //创建之前
        beforeCreate:function(){
            console.log('before Create')
        },
         //创建之后
        created:function(){
            console.log('create');
        },
        //页面挂载前（未渲染）
        beforeMount:function(){
            console.log('beforeMount');
            console.log(this.$el)
        },
        //页面挂载后
        mounted:function(){
            console.log('Mounted');
            console.log(this.$el);
        },
        //当 vm.$destroy()调用时
        beforeDestroy:function(){
            console.log('beforeDestyor')
        },
        destroyed:function(){
            console.log('destroyed');
        },
        //当数据更改时
        beforeUpdate:function(){
            console.log('beforeUpdate');
        },
        updated:function(){
            console.log('updated');
        }
    })

二. Vue指令

数据绑定：

绑定数据有三种方式，插值，v-model和v-bind。

2.1 v-model

v-model是一个指令，实现数据双向绑定，再表单元素外使用不起作用，限制在<input>、<select>、<textarea>、components中使用 

    <div id="app">
        <input type="text" v-model="inputValue"> //绑定数据inputValue
    </div>
        var app = new Vue({
            el: '#app',
            data: {
                inputValue: ''
            }
        })

2.2 v-bind

v-bind 指令被用来响应地更新 HTML 属性，其实它是支持一个单一 JavaScript 表达式 （v-for 除外）。

其缩写为—- :

  <!-- 完整语法 -->
  <a v-bind:href="url">...</a>

  <!-- 缩写 -->
  <a :href="url">...</a>

2.3 v-on

v-on 指令用来绑定事件。

其缩写为—-@

  <!-- 完整语法 -->
  <button v-on:click="handleBtnClick">提交</button>

  <!-- 缩写 -->
  <button @:click="handleBtnClick">提交</button>
 var app = new Vue({
            el: '#app',
            data: {
            },
            methods: {
                handleBtnClick: function () {
                 //dosomething...
                }
            }
        })

2.4 v-for

v-for 可以绑定数据到数组来渲染一个列表： 

    <div id="app">
        <ul>
            <li v-for="item in list">{{item}}</li>
        </ul>
    </div>
        var app = new Vue({
            el: '#app',
            data: {
                list: [1,2,3,4],
            },
        })

v-for 可以通过一个对象的属性来迭代数据： 

<div id="app">
  <ul>
    <li v-for="value in object">
    {{ value }}
    </li>
  </ul>
</div>

<script>
new Vue({
  el: '#app',
  data: {
    object: {
      name: 'wukai',
      age: 18
    }
  }
})
</script>
输出： 
    wukai
    18

你也可以提供第二个的参数为键名： 

<div id="app">
  <ul>
    <li v-for="(value, key) in object">
    {{ key }} : {{ value }}
    </li>
  </ul>
</div>
输出：
    name : wukai
    age : 18

第三个参数为索引： 

<div id="app">
  <ul>
    <li v-for="(value, key, index) in object">
     {{ index }}. {{ key }} : {{ value }}
    </li>
  </ul>
</div>
输出：
    0. name : wukai
    1. age : 18

v-for实现列表渲染，key值唯一且不为index可以最大化提高性能

    <div id="app">
        <div v-for="(item,index) of list" :key="item.id">
            {{item.text}}----{{index}}
        </div>
    </div>
var vm=new Vue({
 el:"#app",
 data:{
     list:[
         {id:"AKL001",text:"Wukai"},
         {id:"AKL002",text:"XXXXX},
         {id:"AKL003",text:"XXXXXXXX"}
    ]
 }
})

直接操作数组改变数组某一项的值不会使页面改变，若想改变需要使用数组变异方法： 

push, pop,shift, unshift, splice, sort, reverse

使用Set方法修改对象或数组，让页面发生变化: 

Vue.set(vm.obj,"property","value")

2.5 v-if / v-else

v-if的值为’true’时显示则v-else不显示，两个标签要挨着写。

 <div id="app">
         <p v-if="show">v-if</p>
      <p v-else>else</p>
 </div>
var vm=new Vue({
        el:"#app",
        data:{
            show:false,
             }
})

默认条件渲染会复用标签，增加key值之后就会取消标签的复用。

 <div id="app">
     <div v-if="show">
        用户名：<input key="username"/>
     </div>
        <div>
        邮箱：<input key="password"/>
     </div>
 </div>

三. 计算属性和侦听器

3.1 计算属性

3.1.1 计算属性

模板内的表达式非常便利，但是设计它们的初衷是用于简单运算的。在模板中放入太多的逻辑会让模板过重且难以维护。例如： 

<div id="example">
  {{ message.split('').reverse().join('') }}
</div>

在这个地方，模板不再是简单的声明式逻辑。你必须看一段时间才能意识到，这里是想要显示变量 message 的翻转字符串。当你想要在模板中多次引用此处的翻转字符串时，就会更加难以处理。

所以，对于任何复杂逻辑，你都应当使用计算属性。

<div id="example">
  <p>Computed fullName: "{{ fullName }}"</p>
</div>
var vm = new Vue({
  el: '#example',
  data:{
        firstName:'Wu',
        lastName:'Kai'
    },
  //computed 计算属性
  computed:{   
          //计算属性的getter
        fullName:function(){
            // `this` 指向 vm 实例
            return this.firstName+this.lastName;
        }
    }
  })

你可以打开浏览器的控制台，自行修改例子中的 vm。vm.fullName 的值始终取决于 vm.firstName和vm.lastName 的值。

Vue 知道 vm.fullName 依赖于 vm.lastName和vm.firstName，因此当 vm.firstName 发生改变时，所有依赖 vm.fullName 的绑定也会更新。

计算得到的属性的，计算执行一次后，将结果存储在缓存中，如果依赖属性没有变化，则不再执行计算，性能高。

3.1.2 计算属性的get和set

计算属性默认只有 getter ，不过在需要时你也可以提供一个 setter ： 

 //计算属性
        computed: {
            fullName: {
                get: function() {
                    return this.firstName + this.lastName;
                },
                //当值被重置时，获得设置值
                set: function(value){
                    var arr=value.split(' ');
                    this.firstName=arr[0];
                    this.lastName=arr[1];
                }
            }
        }

3.2 侦听器

监听某一属性，当监听的属性值发生变化之后，触发函数。

虽然计算属性在大多数情况下更合适，但有时也需要一个自定义的侦听器。这就是为什么 Vue 通过 watch 选项提供了一个更通用的方法，来响应数据的变化。当需要在数据变化时执行异步或开销较大的操作时，这个方式是最有用的。 

    data:{
        firstName:'Wu',
        lastName:'Kai',
        fullName:"WuKai",
    },
    watch:{
        firstName:function(){
            this.fullName=this.firstName+this.lastName;
        },
        lastName:function(){
            this.fullName=this.firstName+this.lastName;
        }
    }

以上代码，当firstName或者lastName改变时，触发函数，相应的改变fullName的值。

3.3 方法vs计算属性

我们可以通过在表达式中调用方法来达到同样的效果：

<p>fullName: "{{ fullName() }}"</p>
      //在组件中
    data:{
        firstName:'Wu',
        lastName:'Kai'
    },
    methods:{
        fullName:function(){
            return this.firstName+this.lastName;
        }
    }

我们可以将同一函数定义为一个方法而不是一个计算属性。两种方式的最终结果确实是完全相同的。

然而，不同的是计算属性是基于它们的响应式依赖进行缓存的。

只在相关响应式依赖发生改变时它们才会重新求值。这就意味着只要 firstName 还没有发生改变，多次访问 fullName 计算属性会立即返回之前的计算结果，而不必再次执行函数。

四. Class与Style绑定

我们可以用b-bind 绑定Dom元素的Class或者Style，并且绑定的表达式类型除了字符串之外，还可以是对象或者数组。

4.1 Class绑定

4.1.1 绑定一个对象

我们可以传给 v-bind:class 一个对象，以动态地切换 class： 

 .redFontColorClass{
     color: red;
     font-size: 200px;
     font-weight: 800;
 }
<div id="app" :class="{redFontColorClass: IsActivated}" >
        Hello wk
</div>
var vm=new Vue({
    el:"#app",
    data:{
        IsActivated:false,
    }
})

上面的语法表示 redFontColorClass 这个 class 存在与否将取决于数据属性 IsActivated 的 truthiness(真值)

4.1.2 与普通class属性并存

v-bind:class 指令也可以与普通的 class 属性共存。当有如下模板: 

<div
  class="static"
  v-bind:class="{ active: isActive, 'text-danger': hasError }"
></div>
data: {
  isActive: true,
  hasError: false
}

结果渲染为：

当 isActive 或者 hasError 变化时，class 列表将相应地更新。例如，如果 hasError 的值为 true，class 列表将变为 "static active text-danger"。

4.1.3 绑定的对象定义在data中

绑定的数据对象不必内联定义在模板里： 

<div v-bind:class="classObject"></div>
data: {
  classObject: {
    active: true,
    'text-danger': false
  }
}

4.1.4 绑定一个计算属性

并且可以绑定一个对象的计算属性：

<div v-bind:class="classObject"></div>
data: {
  isActive: true,
  error: null
},
computed: {
  classObject: function () {
    return {
      active: this.isActive && !this.error,
      'text-danger': this.error && this.error.type === 'fatal'
    }
  }
}

4.1.5 绑定一个数组：

 <div id="app" :class="[redFontClass]"
                  @click="changeFontColor"
    >
        Hello World
  </div>
var vm=new Vue({
    el:"#app",
    data:{
        redFontClass:""
    },
    methods:{
        changeFontColor:function(){
            /*
            点击绑定的元素时，如果这个元素的redFontClass为redFont这个class时
            将redFontClass的样式切换为""(即去除样式)
            否则将redFontClass的class切换为 redFont这个class
            */
            if(this.redFontClass==="redFont"){
                this.redFontClass=""
            }else{
                this.redFontClass="redFont"
            }
        }
    }
})

4.2 Style绑定

v-bind:style 的对象语法十分直观——看着非常像 CSS，但其实是一个 JavaScript 对象。CSS 属性名可以用驼峰式 (camelCase) 或短横线分隔 (kebab-case，记得用引号括起来) 来命名： 

<div v-bind:style="{ color: activeColor, fontSize: fontSize + 'px' }"></div>
data: {
  activeColor: 'red',
  fontSize: 30
}

4.2.1 绑定一个对象

直接绑定到一个样式对象通常更好，这会让模板更清晰： 

<div v-bind:style="styleObject"></div>
data: {
  styleObject: {
    color: 'red',
    fontSize: '13px'
  }
}

同样的，对象语法常常结合返回对象的计算属性使用。

4.2.1 绑定一个数组

v-bind:style 的数组语法可以将多个样式对象应用到同一个元素上： 

<div v-bind:style="[baseStyles, overridingStyles]"></div>
